利用GitHub搭建一个个人博客

1.准备条件

  • GitHub账号

  • 域名（非必须）

2.新建GitHub仓库

首先在GitHub任何页面右上角，使用+下拉菜单选择New repository

这里的仓库名称设置为 username.github.io，一定是你的用户名+github.io，例如我的用户名是xw6666，所以我建立的仓库名为xw6666.github.io
